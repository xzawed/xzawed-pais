# Architecture Decision Records (ADR)

아키텍처 결정 기록. 중요한 설계 결정과 그 근거를 문서화한다.

## 형식

각 ADR은 다음 구조를 따른다:

- **날짜**: YYYY-MM-DD
- **상태**: 제안 | 채택 | 폐기
- **컨텍스트**: 결정이 필요했던 배경
- **결정**: 무엇을 결정했는가
- **근거**: 왜 이 결정을 했는가
- **결과**: 이 결정으로 인해 발생하는 것
